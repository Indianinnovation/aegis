The Problem

Power shouldn't come
at the cost of safety.

OpenClaw proved people want autonomous AI agents. Aegis proves you can have them without gambling your company's data.

✕Shell commands run directly on host machine — one prompt injection = catastrophe
✕Memories stored in plaintext — full file access = full memory access
✕No audit trail — you can't know what your agent did or when

Core Pillars

Four layers.
Zero compromises.

01 / 04
🏖️

Docker Sandbox

Every tool call executes in an isolated, ephemeral container. No network egress. No filesystem writes. All Linux capabilities dropped. Root access impossible. The host machine is never touched.

--network=none --cap-drop=ALL 30s timeout 256MB limit
02 / 04
🛡️

OPA Policy Engine

Open Policy Agent evaluates every tool call before execution. Dangerous shell patterns, sensitive file paths, and destructive SQL are blocked at the policy layer — not by the LLM.

Allowlist model Pattern blocking Human-in-loop Risk levels
03 / 04
🔒

Encrypted Memory

Agent memories are encrypted at rest using AES-256 with per-user keys derived via scrypt. Cross-user reads are cryptographically impossible. GDPR right-to-erasure is a single command.

AES-256 scrypt key derivation GDPR erasure Zero PII in logs
04 / 04
📋

Audit Trail

Every agent action is recorded in a tamper-evident JSONL audit log. Hashed user identities. Hashed arguments. Decision reason codes. Ships directly to Splunk, Datadog, or CloudWatch.

JSONL format SIEM-ready SHA-256 identities SOC2 aligned

Architecture

Secure by design,
not by accident.

Every request flows through four independent security layers before execution. Any layer can block independently.

Gateway
mTLS + JWT Auth + Rate Limiting — All channel traffic authenticated. 30 req/min per user. Nginx reverse proxy with TLS termination.
↓
Policy
OPA Policy Engine — Every tool call evaluated against allowlist before execution. Dangerous patterns, sensitive paths, and destructive operations blocked at policy layer.
↓
Agent
LangGraph Agent + Tool Execution — Reasoning layer with tool calls, sandboxed execution, streaming responses, and human-in-the-loop confirmation for high-risk actions.
↓
Persistence
Encrypted Memory + Audit Log — AES-256 memory store with per-user keys. Every action written to tamper-evident JSONL. HashiCorp Vault for all secrets.
